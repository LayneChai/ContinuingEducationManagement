# 启智继续教育管理系统

启智继续教育管理系统是一个面向继续教育、职业培训与院校进修场景的在线教育管理平台。项目围绕“课程学习 - 学时统计 - 考试测评 - 作业提交 - 证书发放 - AI 辅导”构建完整业务闭环，采用前后端分离架构，支持管理员、教师、学生三端协同。

## 项目亮点

- 面向继续教育真实业务链路，覆盖课程、学习、考试、作业、证书全流程
- 三端角色独立，支持管理员、教师、学生分角色访问与菜单动态分发
- 内置 AI 辅导能力，支持基于 DeepSeek 的课程上下文流式对话
- 前后端均已具备可运行骨架与核心联调页面，适合课程设计、项目展示与后续扩展

## AI 配置

项目中的 AI 对话依赖 DeepSeek。为了安全起见，请使用环境变量注入，不要将密钥写入代码、配置文件或提交到仓库。

示例：

```bat
set DEEPSEEK_API_KEY=your_deepseek_key
start-backend.bat
```

常用环境变量：

- `DEEPSEEK_API_KEY`
- `DEEPSEEK_BASE_URL`
- `DEEPSEEK_MODEL`
- `DB_URL`
- `DB_USERNAME`
- `DB_PASSWORD`
- `REDIS_HOST`
- `REDIS_PORT`
- `REDIS_DATABASE`

更多示例见：`ENVIRONMENT.example.md`

## 建议演示路径

为了展示完整业务闭环，建议按以下顺序演示：

1. 管理员登录，创建教师与学生账号
2. 教师创建课程、章节与课时，并提交审核
3. 管理员审核课程并通过
4. 学生报名课程并记录学习进度
5. 教师创建考试与作业
6. 学生参加考试、提交作业
7. 教师批改作业，学生申请证书
8. 管理员审核证书，演示 AI 辅导与教师 AI 记录查看
